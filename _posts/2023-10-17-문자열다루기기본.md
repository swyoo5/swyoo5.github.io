문자열 길이가 4 또는 6이고 숫자로만 구성돼있는지 확인하는 함수를 작성하는 문제이다.



나의 풀이

```python
def solution(s) :
    if len(s) == 4 or len(s) == 6 :
        if s.isdigit() :
            return True
        else :
            return False
	else :
        return False
```



다른 사람의 풀이

```python
def solution(s) :
    return s.isdigit() and (len(s) == 4 or len(s) == 6)
```
