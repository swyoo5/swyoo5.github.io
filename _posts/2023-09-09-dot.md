* 1차원 x 1차원

  input 배열이 둘 다 1차원일 경우, 내적연산이 이루어진다.

  ```python
  A = np.array([1, 2, 3])
  B = np.array([4, 5, 6])
  print(np.dot(A, B)) 
  # 결과 32
  ```

* 2차원 x 2차원

  행렬의 곱이 된다.

  ```python
  A = np.array([[1, 2], [3, 4]])
  B = np.array([[5, 6], [7, 8]])
  print(np.dot(A, B))
  # 결과
  # [[19 22]
  #  [43 50]]
  ```

* n차원 x 스칼라

  각 원소에 스칼라배를 한다.

  ```python
  A = np.array([[1, 2], [3, 4]])
  b = 2
  print(np.dot(A, b))
  # 결과
  # [[2 4]
  #  [6 8]]
  ```

* n차원 x  1차원

  A의 각 행마다 B와 내적을 해서 반환한다.

  ```python
  A = np.array([[1, 2], [3, 4]])
  B = np.array([2, 5])
  print(np.dot(A, B))
  # 결과
  # [12 26]
  ```

* n차원 x m차원

  A와 B의 첫번째 2차원 배열, A와 B의 두번째 2차원 배열을 곱해서 반환한다.

  ```python
  A = np.array([[1, 2], 
                [3, 4]])
  B = np.array([[[1, 1], 
                 [0, 1]], 
                [[0, 0], 
                 [0, 0]]])
  print(np.dot(A, B))
  # 결과
  # [[[1 3]
  #  [0 0]]
  
  # [[3 7]
  #  [0 0]]]
  ```
