문제는 쉬웠지만 다른 사람의 풀이들이 기가 막혀서 포스팅한다.

내 풀이

```python
def solution(a, b):
    answer = 0
    for idx in range(len(a)) :
        answer += a[idx] * b[idx]
    return answer
```



다른 사람의 풀이1

```python
def solution(a, b) :
    return sum([x * y for x, y in zip(a, b)])
```



다른 사람의 풀이2

```python
solution = lambda x, y : sum(a * b for a, b in zip(x, y))
```



람다와 리스트 컴프리헨션으로 작성한 한줄 정답을 볼 때마다 기가 막히고 코가 막히는 느낌이다.
