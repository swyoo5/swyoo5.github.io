10진법 수를 3진법으로 만든 뒤 뒤집어서 10진법으로 출력하는 문제



```python
def solution(n):
    trinary = ""
    while (n != 0) :
        trinary = str(n % 3) + trinary
        n //= 3
    
    answer = 0
    for i in range(len(trinary)) :
        if trinary[i] == '0' :
            continue
        elif trinary[i] == '1' :
            answer += 3 ** i
        else :
            answer += 2 * 3 ** i
            
    return answer
```



다른 사람의 풀이

```python
def solution(n) :
    tmp = ""
    while n :
        tmp += str(n % 3)
        n = n // 3
        
	answer = int(tmp, 3)

```
