처음 풀이(정답)

```python
def solution(num_list):
    end = 0
    if num_list[-1] > num_list[-2] :
        end = num_list[-1] - num_list[-2]
    else :
        end = num_list[-1] * 2
    answer = num_list
    answer.append(end)
    return answer
```



처음 풀어서 정답을 받고 이정도는 나도 코드 길이를 줄여볼 수 있을것 같아서 줄여봤다.



추가 풀이

```python
def solution(num_list):
    answer = num_list
    num_list.append(num_list[-1] - num_list[-2] if num_list[-1] > num_list[-2] else num_list[-1] * 2)
    return answer
```
