두 수룰 입력받아 두 수의 최대공약수와 최소공배수를 반환하는 함수를 구하는 문제

학부시절 배웠던 유클리드 알고리즘을 복습해보자

```python
def solution(n, m) :
    mul = m * n
    while n != 0 :
        temp = m % n
        m, n = n, temp
    gcd = m
    lcm = mul / m
    answer = [gcd, lcm]
    return answer
```
