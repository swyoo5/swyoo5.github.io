리스트의 인덱스와 그 값을 모두 사용해야 하는 경우가 있으면 enumerate를 사용하면 되는데 습관이 안돼있어 자꾸 깜빡한다.



enumerate 사용 풀이

```python
def solution(food) :
    answer = ""
    for i, num in enumerate(food[1 : ]) :
        answer += str(i) * (num // 2)
    answer = answer + '0' + answer[::-1]
    
    return answer
```
