백준 24267번 설명

https://www.acmicpc.net/problem/24267

```
MenOfPassion(A[], n) {
	sum <- 0;
	for i <- 1 to n-2
		for j <- i + 1 to n - 1
			for k <- j + 1 to n
				sum <- sum + A[i] x A[j] x A[k];
	return sum;
}
```

여기서 반복문이 몇번 실행되는가를 구하는 문제인데, 시그마 연산으로 계산하면 된다.
