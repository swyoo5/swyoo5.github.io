cv2 라이브러리를 설치하려고

```cmd
pip install cv2
```

이 명령어를 터미널에서 실행했지만

![image-20250116140013482](/images/2025-01-16-cv2라이브러리/image-20250116140013482.png)

이런 오류가 발생했다.

대신

```cmd
pip install opencv-python
```

이 명령어를 실행하면 된다.
